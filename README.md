# Sahayak
Sahayak is a student grievence redressal platform. It helps student to post their queries to orgranizational heads.

<h5>Getting started</h5>
</br>
Move to server folder
Create an .env file add your DB_URL and PORT

# How to run server ?
</br>
cd server
</br>
<code>npm i</code>
</br>
<code>npm start</code>

# How to run client ?
</br>
cd client
</br>
<code>npm i</code>
</br>
<code>npm start</code>
