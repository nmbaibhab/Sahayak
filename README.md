# Sahayak
Sahayak is a student grievence redressal platform. It helps student to post their queries to orgranizational heads.

# Getting started
</br>
<h3>Move to server folder</h3>
</br>
<h3>Create an .env file add your DB_URL and PORT </h3>

# How to run server ?
</br>
cd server
</br>
<code>npm i</code>
</br>
<code>npm start</code>

# How to run client ?
</br>
cd client
</br>
<code>npm i</code>
</br>
<code>npm start</code>
